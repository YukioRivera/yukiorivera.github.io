Key Findings
  • ICA provides the best performance with a 3.44% improvement in F1 score while reducing dimensions by 86.74%.
  • Random Projection achieves impressive results with 97.88% dimension reduction while still improving performance by 1.43%.
  • UMAP excels at visualization but shows performance degradation for predictive modeling with only 2 dimensions.
  • All dimensionality reduction methods improved training time, with ICA providing the best balance of performance and efficiency.

Technical Implementation

This project implements various unsupervised learning algorithms and dimensionality reduction techniques on the Spotify 2023 dataset. The analysis includes clustering, dimensionality reduction, and neural network performance evaluation.

Data Preprocessing Example:


# Preprocess dataset for clustering
def preprocess_data(df):
    # Separate target
    y = df['high_playlist_inclusion']
    X = df.drop(columns=['high_playlist_inclusion'])
    
    # Identify categorical columns
    categorical_cols = X.select_dtypes(include=['object']).columns.tolist()
    
    # Create preprocessing pipelines
    numeric_features = X.select_dtypes(include=['int64', 'float64']).columns.tolist()
    
    # Create preprocessing steps
    numeric_transformer = StandardScaler()
    categorical_transformer = OneHotEncoder(drop='first')
    
    # Create column transformer
    preprocessor = ColumnTransformer(
        transformers=[
            ('num', numeric_transformer, numeric_features),
            ('cat', categorical_transformer, categorical_cols)
        ]
    )
    
    # Create and apply preprocessing pipeline
    pipeline = Pipeline(steps=[
        ('preprocessor', preprocessor)
    ])
    
    # Fit and transform data
    X_processed = pipeline.fit_transform(X)
    
    # Split data
    X_train, X_test, y_train, y_test = train_test_split(
        X_processed, y, test_size=0.2, random_state=42
    )
    
    return X_train, X_test, y_train, y_test, X_processed, y

Methodology Insights

This project demonstrates several important insights about unsupervised learning and dimensionality reduction:

Key Findings:

  • Clustering algorithms show different strengths: K-means performs better on well-separated, spherical clusters, while EM can adapt to different cluster shapes and densities.
  • Dimensionality reduction can improve performance: Reducing dimensions not only speeds up computation but can also improve model performance by removing noise.
  • Method selection matters: Different reduction techniques preserve different aspects of the data, affecting downstream performance in different ways.
  • Visualization reveals patterns: Low-dimensional projections can reveal insights about data structure that are hidden in high-dimensional space.
